Setting Up Spark & Spark Notebook

For Windows Users:
  • The most trouble-free way to go is to run Ubuntu via VirtualBox, however you can try the following first
For Mac Users:
  • No need for VirtualBox unless you want to.

Instructions for setting Apache Spark and run in local mode


 Step Task    Command  Expected Result
 1 Download Apache Spark click here to download Spark 1.6.0 for Hadoop 2.6 or later 
 2Create a directory called ucsc-spark in your home directory (or wherever you like)

Copy the Spark distribution file to ucsc-spark		mkdir ~/ucsc-spark
 cd ~/ucsc-spark

mv ~/Downloads/spark-1.6.0-bin-hadoop2.6.tgz ~/ucsc-spark		 
 3 Extract the Spark distribution file tar -xzvf spark-1.6.0-bin-hadoop2.6.tgzshould see directory 
spark-1.6.0-bin-hadoop2.6
 4 Start Spark shell in local mode cd spark-1.6.0-bin-hadoop2.6

./bin/spark-shell		 Should see scala> prompt at the end

To change log level to make it less verbose:

  1. Assuming the current directory is spark-1.6.0-bin-hadoop2.6
  2. cp ./conf/log4j.properties.template ./conf/log4j.properties
  3. Modify ./conf/log4j.properties file
    • Change line log4j.rootCategory=INFO, console to log4j.rootCategory=WARN, console 
  4. Save file and restart Spark shell

Instructions for setting Spark notebook

  • You can download it from http://spark-notebook.io/ or follow this link - http://spark-notebook.io/dl/zip/0.6.2/2.11/1.6.0/2.7.1/true/true.
  • Move the zip file into ~/ucsc-spark and unzip it
  • Since the unzipped folder name is kind of long, you may want to rename it to something shorter like spark-notebook
  • Go into "spark-notebook" folder
  • To startup spark-notebook server, the command ./bin/spark-notebook
    • For Windows users, if the above command didn't work, you may want to try 
      • ./bin/spark-notebook -Dmanager.tachyon.enabled=false
  • Point your browser to http://localhost:9000
To start out with something simple, click on the core folder to explore a few examples.

Enjoy!!
